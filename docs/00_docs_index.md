# 00. Documentation Index

This folder contains the complete documentation package for the Retail Analytics End-to-End Data Pipeline project.

1. `business_requirements.md`
   - Project scope, functional requirements, and success criteria.

2. `data_dictionary.md`

5. `table_mapping_raw_silver_gold.md`
